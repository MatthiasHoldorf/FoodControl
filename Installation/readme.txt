***********************************************
* Anleitung zur Installation von FoodControl: *
***********************************************

Voraussetzungen (werden über den Installer installiert):
- .NET-Framework 4.5
- LocalDB v11.0

***********************************************

1. "setup.exe" ausführen.
2. Falls kein .NET-Framework 4.5 installiert ist, wird dies nun vorab vom Installer übernommen.
3. Installation der Anwendung FoodControl - keine Eingaben sind nötig.
4. Nach Abschluss öffnet sich ein kleines Fenster - es wird automatisch erkannt, ob ein 32-bit oder 64-bit Betriebssystem installiert ist.
   Bitte die automatische Auswahl prüfen und anschließend den Button "install localdb" klicken.
5. Der Installationsassistent für LocalDB v11.0 startet - keine Eingaben sind nötig, lediglich die Installation ist abzuschließen.
6. Nachdem LocalDB installiert wurde, kann die Anwendung FoodControl gestartet werden.

***********************************************

HINWEIS: Der erste Start kann etwas länger dauern, ggf. Prozess beobachten!
